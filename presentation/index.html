Introduction

Sajojo adalah sebuah sistem palang kereta otomatis bertenaga sinar matahari yang sustainable. Dengan menggunakan sensor infra-merah, Sajojo akan mendeteksi adanya kereta api yang akan melintas lalu akan menggerakan motor untuk menutup dan membuka palang kereta.

General Architecture

General architecture of Sajojo

Custom Kernel

Details on Sajojo's Custom Kernel

Untuk booting yang lebih cepat, konfigurasi kernel pada Raspberry Pi dimodifikasi sehingga modul - modul yang tidak diperlukan tidak perlu di-load oleh sistem. Berikut adalah modul yang dimodifikasi dari konfigurasi kernel:

-Hardware driver selain USB, GPIO, mouse, keyboard receiver dihapus
-Mematikan support Raspberry Pi untuk touchscreen dan sound
-Kompresi modul kernel yang tidak digunakan
-Mematikan verbose debugging oleh module
-Support bahasa selain bahasa Inggris dihapus

Bootscript

Details on Sajojo's Bootscript

-Membuat sebuah script yang berisikan perintah untuk run program utama
-Script disimpan di dalam direktori init.d, semua script yang tersimpan pada direktori tersebut akan dijalankan ketika proses boot.

GPIO Service Development

GPIO digunakan dalam berkomunikasi dengan stepper motor, LED serta sensor
